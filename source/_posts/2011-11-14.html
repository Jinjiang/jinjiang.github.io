--- title: "[不老歌] 给我的博客换了几个Mac字体" date: 2011/11/14 13:30 updated: 2011/11/14 13:30 ---

本文摘自 勾三股四 更早时期的 不老歌 博客。

不老歌的默认字体一直是 "Courier New" 对于经常写代码的人来说，这种等宽字体非常亲切，看着也很舒服，不觉得费眼。

后来，我发现比较高版本的Windows里，有另外一个等宽字体，感觉比较漂亮，那就是Consolas，于是我长期把我的博客默认字体设为：

body, td, input, button, select, textarea {
    font-family: Consolas, "Courier New";
}

最近玩了玩Mac，发现Mac下的字体库非常棒！有很多不错的字体，觉得不用一用可惜了，于是在我的博客的CSS里，加入了一些字体选择：
body, td, input, button, select, text area {
    font-family: Optima, Consolas, "Courier New";
}
code, pre {
    font-family: Monaco, Consolas, "Courier New";
}

如果用Mac或iOS设备访问我的博客，会看到比较不一样的字体。也希望在大家看来，是更好看的字体。

另外上周末为 WebReBuild.ORG 准备了一个前端技术的分享，在设计幻灯演示的时候，也一直在纠结字体。我发现在Mac下的那个叫做“字体库”的应用程序非常贴心，可以很方便的浏览Mac操作系统下的所有可用字体，并且按照不同的需求进行了分类，还可以预览不同字号、应用到不同文字上的样子，这让我更有兴趣把所有的Mac字体都看过一遍——要知道在Windows里，浏览一种字体是一件很不方便的事情。

最后推荐几个我在Mac字体库里淘出来的比较对我胃口的字体，如果你的电脑也是Mac系统或iOS系统，不妨也来一起鉴赏一下：
(如果您是通过人人网、鲜果、Google Reader等网站查看下面的内容，可能看不出啥效果来，请移步到原文链接)
Monaco
http://bulaoge.net/?g3g4
个人感觉比较“可爱”的一款等宽字体
Gill Sans
http://bulaoge.net/?g3g4
比较有现代感——事实上，这款字体正是被归类为“现代”类字体，网上见到不少Keynote文档在使用这个字体，不过它被加粗的时候，就不太好看了……
Futura
http://bulaoge.net/?g3g4
这款字体跟上一个Gill Sans感觉差不多，加粗后的效果顺眼一些
Optima
http://bulaoge.net/?g3g4
就是现在我们在Mac下看到的字体，当然可能以后我的博客的字体还会变就是了，上述三个字体都是“现代”类的字体
Lucida Grande
http://bulaoge.net/?g3g4
最后一个推荐的字体，它虽然不在任何子分类里，却非常好看非常常见——因为它是Mac OS的默认字体
