0. 该插件是在 明城 的“使用 Markdown 解析器”的基础上完善的。
1. 使用 <a href="http://daringfireball.net/projects/markdown/" target="_blank">Markdown 语法</a> 编写和发布文章。
2. 在正文上方加入了简单的格式按钮；
3. tab键自动输入4个空格；
4. 正文下方加入了预览功能；
5. 在侧边栏加入了语法提示。

下面是三张截图，一个是工具栏，一个是预览区域，一个是语法提示区域
